Чек-лист для проверки функции find_roots (ax^2 + bx + c = 0)

1. Два различных действительных корня (положительный дискриминант)
   a=2, b=1, c=-3 → ожидается [-1.5, 1.0]

2. Один корень (дискриминант равен нулю)
   a=4, b=-4, c=1 → ожидается [0.5]

3. Отрицательный дискриминант (нет действительных корней)
   a=3, b=2, c=4 → ожидается []

4. Линейное уравнение (a=0, b≠0)
   a=0, b=7, c=-14 → ожидается [2.0]

5. Противоречие (a=0, b=0, c≠0)
   a=0, b=0, c=10 → ожидается []

6. Тождество (a=0, b=0, c=0)
   a=0, b=0, c=0 → ожидается None

7. Дробные коэффициенты
   a=0.25, b=0.5, c=-2 → ожидается [-4.0, 2.0]

8. Очень большие числа (проверка переполнения)
   a=1e6, b=-1e6, c=0 → ожидается [0.0, 1.0]
